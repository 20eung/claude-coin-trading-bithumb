# Release v0.1.0

**Release Date:** 2026-02-26

## Highlights

### Initial Release

Claude Code-based cryptocurrency automated trading system.

### Implemented Features

- **Exchange Integration:** Bithumb API (price lookup, portfolio, trading)
- **Technical Analysis:** RSI, SMA, MACD, Bollinger Bands
- **Market Sentiment:** Fear & Greed Index, News sentiment analysis
- **Automation:** cron scheduling (8-hour interval)
- **Notifications:** Telegram Bot for real-time alerts

### Scripts

| Script | Description |
|--------|-------------|
| `collect_market_data.py` | Bithumb market data + technical indicators |
| `collect_fear_greed.py` | Fear & Greed Index |
| `collect_news.py` | Tavily news collection |
| `capture_chart.py` | Playwright chart capture |
| `get_portfolio.py` | Portfolio lookup |
| `execute_trade.py` | Trade execution with safety guards |
| `notify_telegram.py` | Telegram notifications |
| `run_analysis.sh` | Analysis pipeline |
| `cron_run.sh` | Cron wrapper |
| `setup_cron.sh` | Cron registration helper |

### Claude Skills (6)

- `crypto-trader` - Main orchestration
- `bithumb-api` - Bithumb REST API
- `fear-greed-index` - Fear & Greed Index
- `tavily-news` - News collection
- `chart-capture` - Chart capture
- `trade-notifier` - Telegram notifications

### Safety Guards

| Parameter | Default | Description |
|-----------|--------|-------------|
| `DRY_RUN` | `true` | Simulation only, no actual trading |
| `MAX_TRADE_AMOUNT` | `100,000 KRW` | Per-trade limit |
| `EMERGENCY_STOP` | `false` | Emergency stop all trades |
