# Release v1.0.0

**Release Date:** 2026-03-03

## Highlights

### Exchange Migration: Upbit to Bithumb

Full migration from Upbit API to Bithumb API.

**Changes:**
- `collect_market_data.py` - Base URL `api.upbit.com` -> `api.bithumb.com`
- `execute_trade.py` - Order API changed to v2 (`/v2/orders`), parameter `ord_type` -> `order_type`
- `get_portfolio.py` - Added market validation filter using `/v1/market/all` for KRW pairs
- `capture_chart.py` - Changed to embed TradingView widget in local HTML (bithumb.com blocked headless browsers)
- `notify_telegram.py` - Updated examples to reference Bithumb

**Environment Variables:**
- `UPBIT_ACCESS_KEY/SECRET_KEY` -> `BITHUMB_ACCESS_KEY/SECRET_KEY`

**Skills:**
- Removed `upbit-api` skill
- Added `bithumb-api` skill document (Quotation API v1, Exchange API v2, JWT authentication)
- Updated `crypto-trader`, `chart-capture`, `trade-notifier` skills to reference Bithumb
