# Release v1.1.0

**Release Date:** 2026-03-04

## Highlights

### Cron Automation Improvements

- Added PATH settings for Linux environment in `cron_run.sh`
- `setup_cron.sh` now supports interactive interval selection (4h/8h/12h/24h)
- 8-hour interval cron registration and full pipeline test passed

### Telegram Message Format Improvements

- Switched from MarkdownV2 to HTML format (resolves special character escaping issues)
- Added `report` type: structured JSON to newline format for analysis reports
- Added stdin pipe support: `echo JSON | notify_telegram.py report -`
- Updated `run_analysis.sh` to include `report` JSON format specification in prompt

### Telegram Group Topic Support

- Added `message_thread_id` parameter for Telegram group topics
- Requires `TELEGRAM_THREAD_ID` environment variable

### Repository Rename

- GitHub repository: `claude-coin-trading` -> `claude-coin-trading-bithumb`
