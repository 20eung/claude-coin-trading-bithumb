# Release v1.2.0

**Release Date:** 2026-04-11

## 주요 변경사항

### 데이터베이스: Supabase PostgreSQL → 로컬 SQLite

프로젝트에서 원격 데이터베이스 서비스 의존성을 제거하고 로컬 SQLite 데이터베이스를 사용합니다.

**변경사항:**
- `database/schema.sql` - 6개 테이블의 SQLite 스키마 추가
- `database/db.py` - 로컬 데이터베이스 CRUD 모듈 추가
- `scripts/run_analysis.sh` - Supabase API에서 SQLite 조회로 변경
- **자동 초기화:** DB 파일이 없으면 자동으로 생성

**데이터베이스 테이블:**
- `decisions` - 매매 결정 기록
- `portfolio_snapshots` - 포트폴리오 스냅샷
- `market_data` - 시장 데이터 기록
- `feedback` - 사용자 피드백
- `execution_logs` - 실행 로그
- `strategy_history` - 전략 변경 이력

### 텔레그램 리포트 포맷 개선

- FGI 분류를 한국어로 표시 (Extreme Fear → 극도공포)
- `market` 및 `portfolio` 필드 구조화하여 가독성 향상

**변경 전:**
```
108,260,000: KRW
SMA(20) 103,685,200 (+4.41%)
RSI 58.68
FGI 15 (Extreme Fear)
```

**변경 후:**
```
- 현재가: 108,260,000원
- SMA(20): 103,685,200원 (+4.41%)
- RSI: 58.68
- FGI: 15 (극도공포)
```
